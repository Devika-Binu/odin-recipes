 About the Project
This project focuses on the development of a personal “About Me” Webpage using HTML and CSS. The webpage has been designed to present details about myself and my interests in a structured and visually appealing manner. It demonstrates the effective use of HTML for content organization and CSS for layout design, colour schemes and overall presentation aesthetics.

Skills and Learning
Through the completion of this project, I have gained a comprehensive understanding of the fundamental concepts of web design and development. I have enhanced my ability to create well-structured web pages, apply styling techniques and ensure visual consistency. This project has also strengthened my skills in combining creativity with technical precision to design a user-friendly webpage.
